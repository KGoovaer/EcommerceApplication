## Key Business Rules

| Rule | Summary |
|---|---|
| BR-001 | Customer name and email must be unique |
| BR-002 | Products must belong to exactly one brand and one category |
| BR-003 | Product images must be .jpg/.bmp/.jpeg/.png/.webp ≤ 10 MB |
| BR-004 | Checkout requires a non-empty cart |
| BR-005 | New orders default to status "Processing" |
| BR-006 | Guest cart transfers to order at checkout |
| BR-007 | Customers may cancel their own orders (header only) |
| BR-008 | Admin customer deletion does not cascade to cart/orders |
| BR-009 | Contact enquiries persist until admin removes them |
| BR-010 | Exactly one payment method per order |
